# Module : Scénario "Premiers Pas" pour Argyropée

![Foundry Version](https://img.shields.io/badge/Foundry-v13+-ff6400.svg)
![System](https://img.shields.io/badge/Système-Argyropée-blue.svg)
![Version](https://img.shields.io/badge/Version-1.0.0-brightgreen.svg)

Bienvenue dans la Cité-Franche ! Ce module officiel pour Foundry VTT contient l'intégralité du matériel prêt à jouer pour le scénario d'introduction **"Premiers Pas"** du jeu de rôle *Argyropée*. 

Conçu pour faire gagner un temps précieux au Maître de Jeu, ce module vous évite toute préparation technique et vous permet de vous concentrer sur l'essentiel : l'ambiance et la narration.

---

## 📖 Contenu du Module

Ce module ajoute de nouveaux **Compendiums** (Dossiers de système) à votre monde, contenant l'exhaustivité des rencontres du scénario :

* **Le Casting Complet (Acteurs) :** L'intégralité des Personnages Joueurs (PJs), des dizaines de Personnages Non-Joueurs (PNJs) et créatures du scénario, pré-configurés avec :
  * Leurs statistiques de Santé et de Panache.
  * Leurs bonus d'Initiative calculés.
  * Leurs compétences privilégiées déjà cochées.
  * Leurs magies, équipements et notes de biographie intégrés dans leurs fiches.
* **Les Scènes présentent dans l'ouvrage :** 12 scènes pré-configurés
* **L'intégralité du texte de l'ouvrage écrit par Frédéric Marin**

---

## ⚙️ Prérequis

Ce module est une extension de contenu. Il nécessite impérativement l'installation du système de jeu principal :
* **[Système Argyropée]** (Version 1.0.6 ou supérieure recommandée).

---

## 🚀 Utilisation en jeu

1. Allez dans l'onglet **Compendiums** (l'icône de livre fermé) dans le menu de droite de Foundry.
2. Cherchez le compendium nommé *Acteurs - Premiers Pas*.
3. Vous pouvez glisser les PNJs directement depuis ce compendium vers votre carte, ou faire un clic droit sur le compendium et choisir **"Importer tout le contenu"** pour créer un dossier complet dans votre onglet Acteurs.
